# PicklePlay Admin Guide

This guide is for PicklePlay users who manage platform-wide operations through the admin area.

## Who This Guide Is For

Use this guide if your account is an `Admin` and you need to work on:

- admin console operations
- verifications
- applications
- tournaments
- users
- security-related views
- codes and QR-related tools
- audit and maintenance workflows
- achievements administration

## Important Notes Before You Start

1. Admins usually see more tools than normal users.
2. Admin pages may include functions that do not appear for players, coaches, owners, or managers.
3. Even though admins often bypass feature restrictions, login and security requirements still apply.
4. If 2FA is pending, complete it first before continuing.

## 1. Log In as an Admin

1. Open PicklePlay and log in.
2. Complete email verification if needed.
3. Complete 2FA if prompted.
4. Confirm that your active account has admin access.

## 2. Main Admin Navigation

Common admin areas include:

- `Admin Console`
- `Court Owner Apps`
- `Manage Squads`
- `Manage Achievements`
- `Dashboard`

Depending on the UI, you may also still see profile, news, or other shared sections.

## 3. Use the Admin Dashboard

The admin dashboard helps you review overall platform activity.

### It may include

- high-level admin operations
- monitoring views
- action shortcuts
- broadcast or maintenance-related entry points

### Suggested workflow

1. Open `Dashboard`.
2. Review what needs attention.
3. Move into the correct admin tool from there.

## 4. Use the Admin Console

The `Admin Console` is the main workspace for platform management.

### Common admin console tabs or sections

- overview
- tournaments
- applications
- verifications
- ID training
- users
- security
- codes
- QR codes
- audit
- terms
- website or maintenance settings

### Suggested workflow

1. Open `Admin Console`.
2. Choose the tab that matches your current task.
3. Review pending records first.
4. Process each item carefully.

## 5. Manage Court Owner Applications and Verifications

Open `Court Owner Apps` or the verification-related admin tab when reviewing onboarding or compliance items.

### Common uses

- review pending applications
- review verification records
- approve or reject submissions
- track the status of owner onboarding

### Suggested workflow

1. Open the verification or applications area.
2. Review pending entries first.
3. Check the submitted details.
4. Approve, reject, or update the status as needed.

## 6. Manage Tournaments as an Admin

Admins may have tournament-related tools in the admin area and may also access deeper management pages.

### Common tournament tasks

- review tournament records
- manage tournament state
- approve or inspect tournament-related items
- open deeper tournament management flows when required

## 7. Manage Users and Security

Use the `Users` and `Security` sections for platform oversight.

### These areas may be used to

- inspect user records
- review security-related issues
- work with role-related or policy-related admin tasks

### Suggested workflow

1. Open the relevant section.
2. Review the issue or record carefully.
3. Apply the correct admin action.

## 8. Manage Codes, QR Tools, and Audit

Admins may also have access to:

- code management
- QR-related tools
- audit logs

These sections are useful for broader platform operations and tracking.

## 9. Manage Website or Maintenance Settings

Some admin views include website or maintenance controls.

### These may affect

- feature availability
- user access
- maintenance state
- system-wide visibility of app sections

Use these controls carefully because they can affect many users at once.

## 10. Manage Achievements

Use `Manage Achievements` for achievement-related administration.

### Common uses

- review achievement records
- manage definitions or availability
- support achievement-related platform tasks

## 11. Use Shared Areas When Needed

Admins may also use some shared app areas like:

- profile
- squads
- tournament management pages
- dashboard shortcuts

Because admin access is broader, your view may not match what standard users see.

## 12. Common Admin Problems

### I do not see the expected admin tool

Check the following:

1. You are logged in with the correct account.
2. Admin access is active on the account.
3. 2FA is not still pending.

### My screen does not match a normal user guide

That is expected. Admin access often bypasses normal feature limits and exposes extra tools.

### I can reach an admin page but not the matching user-facing route

Some admin tasks are available only through admin panels or direct admin routes, not through standard user navigation.

## 13. Recommended Daily Admin Workflow

1. Open `Dashboard`.
2. Review urgent platform tasks.
3. Open `Admin Console`.
4. Check verifications and applications.
5. Review tournaments, users, or security items.
6. Check maintenance or website controls if needed.
7. Review achievement or squad-related admin tasks.
